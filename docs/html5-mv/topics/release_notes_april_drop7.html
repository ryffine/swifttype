Release 6 - April drop #7

Resolved Issues

  • Resolved an issue where publishing webhelp from a DITA map resulted in an XPath expression appearing as the webhelp title instead of the map title (DE83).
  • Commented out the deprecated "Footer Text" webhelp string for all languages. The footer text is now specified in a key file (DE87).
  • Resolved an issue where figure and table numbering would get reset at certain point throughout document when the -Dge.add.chapter.prefix=yes -Dge.add.subheader.prefix=yes build parameters are were set. Figure and table numbering now correctly increments regardless of build parameters settings (DE89).
  • Several updates to cross-reference output to align with GE output specifications (DE91).
    • PDF cross-references to sections and examples no longer include a bullet before the link text.
    • PDF and webhelp cross-references to task steps now correctly include the step number in all situations.
    • PDF cross-references to li items now resolve to the correct li element.
  • Updated several PDF and webhelp strings, used for French language output, to use the correct translations (US322).

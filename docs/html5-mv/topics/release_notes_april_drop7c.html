Release 6 - April drop #7c

Resolved Issues

  • Hazardstatement type notice elements in webhelp output are now correctly formatted to match the PDF formatting for both IEC- and ANSI-style admonitions (DE93).
  • Hazardstatement admonitions in the PDF output are now consistently indented regardless of the type (DE94).
  • The mainbooktitle content is now correctly centered within the footer of even pages in the PDF output (US314).
  • Figure titles now appear before the image in PDF and webhelp output (US320, US 326).
  • Multiple fr-FR localization strings were updated in the PDF and webhelp output to address translation and formatting issues (US322).
  • Generated link text for cross-references to figures and tables no longer includes the figure or table title. The inserted text now reads "Figure x" or "Table x", where x is the appropriate figure or table number (US328).

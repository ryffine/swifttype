Release 6 - February drop #1

Resolved Issues

  • PDF output now uses duplex page headers and footers throughout the document (DE52)
  • Updates to the Webhelp navigation pane CSS to prevent the expand/collapse book icons from disappearing when they are clicked. (DE53)
  • Update to webhelp page footer to fix the ge.com link. This issue was caused by the presence of comments within the string variable content. (DE59)
  • Updates to PDF and Webhelp localization strings for en-US and fr-FR. (DE60)
  • Authors now have the option to include a "Doing business as" (DBA) string on the back cover of PDF output by adding a second bookrestriction element to the bookmap with the value set to dba. (DE60)
    <bookrights value="ce">
    <copyrfirst>
        <year><keyword conkeyref="GE_Legal/copyright_year"/></year>
    </copyrfirst>
    <bookowner/>
    <bookrestriction value="reserved"/>
</bookrights>
<bookrights>
    <bookowner/>
    <bookrestriction value="dba"/>
</bookrights>
  • Manufacturer and EC Rep icons will now appear on the back cover of PDF documents. This update restores a feature originally developed as part of Release 5 - July drop #1 . (DE61)
  • Webhelp search results for topics that do not contain a short description element will no longer display Free Web tutorials beneath the search result item link. Search results for topics containing a short description continue to display the short description text in the results list. (DE62)
  • Webhelp output will now display only the contents of mainbooktitle in the masthead. (DE68)

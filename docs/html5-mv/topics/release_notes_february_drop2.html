Release 6 - February drop #2

Resolved Issues

  • Topic breadcrumbs now appear for multiple topic nesting levels (DE58).

    Links appear based on topic nesting level. Maps used to group topics do not influence the topic hierarchy.

  • Webhelp images with the scalefit attribute set to "yes" are now correctly resized to fit within the browser window(DE63).
  • Publishing webhelp output with draft comments enabled now displays the conditional attribute value applied to content in addition to surrounding the content with a red border (DE65).
  • Update the webhelp CSS so that the expand/collapse buttons do not overlap the previous/next navigation buttons (DE66).
  • TOC entries no longer contain a line breaks for ph content (DE67).
  • the scrollbar and content at the bottom of webhelp pages no longer gets cut off (DE69).

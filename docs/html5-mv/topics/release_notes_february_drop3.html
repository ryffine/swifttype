Release 6 - February drop #3

Resolved Issues

  • In webhelp output, outputclass="newtarget" will now cause the link target to open in a new browser window (DE72).
  • Resolved an issue where the webhelp navigation pane scrollbar is cut off at the bottom of the window (DE74).
  • Clicking the webhelp mailto link opens a new message to help.feedback@ge.com unless the author chooses to override the value by specifying an alternate email address in a bookmap/bookmeta/data/data-about/data element with an id attribute set to "biz_email" (DE75).
  • Added a period to the end of "All rights reserved" on the PDF cover page (DE77).
  • Tables that span multiple pages in PDF output now have a "(Continued)" label added to the table header on subsequent pages. (DE78).
  • Addresses on the PDF back cover now correctly display telephone number below the address (DE79).
  • The "Page intentionally left blank" string for French PDF output has been updated to use the correct translation (DE82).

Release 6 - February drop #4

Resolved Issues

  • Resolved an issue where choption elements in choicetables were centered instead of left-aligned when the content was viewed using older browsers, such as Firefox 38 (DE64).
  • Both the PDF and webhelp outputs now display a Draft watermark label on all pages (DE70).
    • For webhelp, you can enable the Draft label by setting the -Dargs.draft=yes build parameter.
    • For PDF, you can enable the Draft label by setting the -Doutput.draft.comment=yes build parameter.
  • Webhelp output now correctly creates expandable/collapsible content (twistys) when outputclass="show_hide_expanded" is set on the content (DE73).

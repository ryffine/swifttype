Release 6 - June drop #10

New features, enhancements, and Resolved Issues

  • Cross-references to task steps, unordered list items, figure callout list items, and cross-references within figure callout descriptions now resolve correctly (DE104, DE116).
  • The Previous topic (<) and Next topic (>) buttons have been removed from the webhelp skin (DE105).
  • Table desc element content now uses normal font weight instead of bold (DE106).
  • In webhelp output, clicking the chapter title now expands and collapses the TOC content. This update restores a behavior that was present in the DITA-OT 1.7.5 webhelp output (DE107).
  • Concept, task, and reference topics nested within another topic are now numbered correctly for PDF output when the chapter and section numbering is enabled using the -Dge.add.chapter.prefix=yes and -Dge.add.subheader.prefix=yes build parameters (DE108).
  • Hazardstatement or note element content within a task step but before the cmd element now renders before the step number label (DE109).
  • IEC hazardstatement content is now correctly indented regardless of whether or not an image is specified (DE110).
  • Resolved an issue where duplicate hazardstatement images were appearing in webhelp output (DE111).
  • Resolved an issue where table cell content was removed from PDF output if the table cell only contained a single item (DE115).
  • Updated the formatting for level 4 and level 5 headings in PDF output to conform to GE specifications. Section title content formatting was also updated to match the formatting of level 5 headings (DE118).

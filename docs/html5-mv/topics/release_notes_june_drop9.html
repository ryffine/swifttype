Release 6 - June drop #9

New features, enhancements, and Resolved Issues

  • In PDF output, images and titles within a fig element now honor the value of the image align attribute. Figure callouts are always left aligned regardless of the image alignment setting (US296).

    The following markup provides an example of how to center a figure within the text column.

    
      <fig>
       <title>A labeled image</title>
       <image href="z600_Front_Back-Test.svg" align="center"/>
       <ol id="ol_f5n_p2d_d1b">
        <li>DVD drive*</li>
        <li>SD card drive</li>
        <li>Power button</li>
        <li>USB inputs</li>
       </ol>
      </fig>
